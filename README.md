# Sheerun's dotfiles

## Requirements and installation

The only requirement are `zsh` and `curl` installed.

```shell
zsh <(curl --location dotfiles.sher.pl/sheerun)
```

I encourage you to fork this repo and replace `sheerun` with your login.
